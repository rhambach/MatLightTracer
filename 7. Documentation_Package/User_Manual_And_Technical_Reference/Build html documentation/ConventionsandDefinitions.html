Starting the Toolbox

Unit for lengths

Optical system has defined Lens and wavelength units. Everything related to optical system will be in those units. They include

  • all surface parameters, optical system parameters,
  • returns of functions operating on optical systems such as rayTracer results, multipleRayTracer results,…

But for other things which are not directly related to the optical system such as

  • Ray objects, returns of getChiefRay, getInitialRayBundle, getMariginalRay,
  • returns of  getPrimaryWavelength, getSystemWavelengths, getSystemFieldPoints,

the SI units are used for both wavelength and lengths.
